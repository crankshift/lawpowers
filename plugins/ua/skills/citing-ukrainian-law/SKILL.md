# citing-ukrainian-law

Стандартизований формат цитування робить правові позиції перевіряними, читабельними та відтворюваними.

## Посилання на нормативний акт

**Перше (повне) згадування:**
```
ч. 1 ст. 257 Цивільного кодексу України
(у редакції від 01.01.2024, zakon.rada.gov.ua/laws/show/435-15)
```

**Повторне в тому самому документі:**
```
ч. 1 ст. 257 ЦК
```

**Дослівна цитата** (обов'язково в лапках):
```
«Загальна позовна давність встановлюється тривалістю у три роки»
(ч. 1 ст. 257 ЦК, редакція чинна з 01.01.2024).
```

## Ієрархія елементів норми

Порядок — від меншого до більшого:
- `пп.` — підпункт
- `п.` — пункт
- `ч.` — частина
- `ст.` — стаття
- `р.` / `гл.` — розділ / глава

**Приклад:** `пп. «а» п. 1 ч. 2 ст. 10 ЗУ «Про...»`.

## Судова практика

**Постанова касаційного суду ВС:**
```
Постанова Касаційного цивільного суду у складі Верховного Суду
від 15.03.2023 у справі № 910/1234/22
(reyestr.court.gov.ua/Review/{doc_id}).
```

**Постанова Великої Палати ВС:**
```
Постанова Великої Палати Верховного Суду
від 15.03.2023 у справі № 910/1234/22.
```

**Рішення КСУ:**
```
Рішення Конституційного Суду України
від 15.03.2023 № 3-р/2023 у справі № 1-15/2023
(ccu.gov.ua).
```

**Рішення ЄСПЛ:**
```
Рішення ЄСПЛ у справі «Іванов проти України»
від 15.03.2023, заява № 12345/20.
```

## Common mistakes

- **Посилання без редакції.** Норма могла змінюватися — цитата без дати редакції ненадійна.
- **Самовільні номери справ ВС.** Без верифікації в ЄДРСР — ризик галюцинації.
- **Плутання КЦС/КГС/КАС/ККС ВС та ВП ВС.** Різні органи, різна обов'язкова сила позицій.
- **Рішення нижчих інстанцій як «правова позиція».** Обов'язкову силу мають позиції ВС. Апеляції/місцеві — лише per curiam у конкретній справі.
- **Цитати ЄСПЛ без заяви і дати.** Ідентифікатори рішення ЄСПЛ — назва справи + номер заяви + дата.
- **Посилання на «постанову Пленуму ВСУ» як чинну.** Пленум ВСУ скасовано реформою 2016; правові позиції тепер формує ВС через касаційні суди і Велику Палату.
