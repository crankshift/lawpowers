# searching-edrsr

ЄДРСР — Єдиний державний реєстр судових рішень. Містить рішення українських судів усіх інстанцій (крім законних винятків). Використовується для перевірки судової практики та верифікації цитат.

## URL patterns

| Ціль | URL |
|---|---|
| Пошук (головна) | `https://reyestr.court.gov.ua/` |
| Документ за реєстр. номером | `https://reyestr.court.gov.ua/Review/{doc_id}` |

**Важливо:** «номер справи» (формат `XXX/NNNN/YY`) ≠ «номер документа в реєстрі» (`doc_id`). Пошук на головній — за номером справи.

## Структура номера справи

`КОД_СУДУ/ПОРЯДКОВИЙ/РІК` — напр. `910/1234/24`:
- `910` — код суду (у прикладі — Господарський суд м. Києва).
- `1234` — порядковий номер у суді.
- `24` — рік реєстрації.

За номером справи можна знайти всі рішення/ухвали по цій справі в усіх інстанціях.

## Пошукові критерії

- **Номер справи** — найнадійніший.
- **Суд** — з випадного списку.
- **Дата ухвалення** — точна або діапазон.
- **Тип документа** — рішення, постанова, ухвала, окрема ухвала.
- **Суддя** — ПІБ (корисно для виділення практики конкретної палати).
- **Ключові слова** — у тексті документа (повнотекстовий пошук).

## Workflow

1. **Є номер справи** → пошук напряму. Отримати ВСІ рухи справи (першої, апеляційної, касаційної інстанцій) — перевірити, чи не скасовано вищою інстанцією.
2. **Немає номера, шукаєш практику** → ключові слова + тип документа («постанова») + період. Пріоритет — касація (ВС).
3. **Шукаєш правову позицію ВС** → почати з supreme.court.gov.ua (позиції згруповано за категоріями), далі — повний текст у ЄДРСР за номером справи.
4. **Верифікація цитати** → завжди відкривати оригінал у ЄДРСР. Не покладатися на фрагменти з коментарів, статей, pdf-підбірок.

## Інстанції та їх пріоритет

| Інстанція | Обов'язкова сила позиції |
|---|---|
| Велика Палата ВС | Висока (виключна юрисдикція по особливо складних) |
| Касаційний цивільний / господарський / адмін. / кримінальний суд ВС | Висока (обов'язкова для всіх судів) |
| Апеляційні суди | Лише для конкретної справи |
| Місцеві суди | Лише для конкретної справи |

## Common mistakes

- **Цитування справ «з пам'яті».** Номери справ без верифікації в ЄДРСР — ризик галюцинації. Завжди перевіряти.
- **Ігнорування руху справи.** Рішення першої інстанції могло бути скасовано апеляцією/касацією — цитата скасованого акта безпідставна.
- **Плутанина КЦС/КГС/КАС/ККС ВС та Великої Палати.** Різні органи ВС. При конфлікті позицій — позиція Великої Палати превалює.
- **Покладання на витяги.** Контекст часто втрачається; читати оригінальний текст.
- **Плутання номерів.** Номер справи ≠ номер документа в реєстрі.
