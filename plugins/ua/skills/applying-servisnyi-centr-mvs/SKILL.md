# applying-servisnyi-centr-mvs

**Сервісні центри МВС** (СЦ МВС) — структурні підрозділи Головного сервісного центру МВС (ГСЦ МВС). З 2015 року МРЕВ реформовано в сервісну модель; з 2019 р. — новий порядок (постанова КМУ № 1122 від 24.12.2019). СЦ МВС надають послуги з реєстрації транспортних засобів, видачі посвідчень водія, оформлення бланкового ДКП ТЗ тощо.

**Основні акти:**
- **Постанова КМУ № 1388 від 07.09.1998** (`1388-98-п`) — Порядок державної реєстрації (перереєстрації), зняття з обліку автомобілів, автобусів... — ключовий акт для ТЗ.
- **Постанова КМУ № 340 від 08.05.1993** (`340-93-п`) — Положення про порядок видачі посвідчень водія.
- **Постанова КМУ № 1122 від 24.12.2019** (`1122-2019-п`) — про реорганізацію СЦ МВС.
- **Наказ МВС № 490 від 31.08.2016** (`z1340-16`) — Інструкція про порядок здійснення адміністративних процедур з реєстрації ТЗ.
- **Наказ МВС № 379 від 08.07.2022** (`z0773-22`) — бланк договору купівлі-продажу ТЗ.
- **ЗУ «Про дорожній рух»** № 3353-XII.

**Джерела:**
- **hsc.gov.ua** — офіційний сайт ГСЦ МВС: запис у чергу, калькулятор зборів, перелік послуг, статус готовності документа.
- **e.hsc.gov.ua** — електронний кабінет.
- **«Дія»** — деякі послуги (замовлення перевірки, запис).
- **Постанови і накази** — `zakon.rada.gov.ua`.

## 1. Запис у чергу

**Спосіб 1 — онлайн:** `https://hsc.gov.ua/e-queue` — вибір СЦ, дати, послуги. Потрібен BankID або e-mail.

**Спосіб 2 — жива черга:** у деяких СЦ; нестабільно в умовах воєнного стану.

**Спосіб 3 — «Дія»:** певні послуги доступні напряму без черги (зміна реєстрації при переоформленні, замовлення довідки).

## 2. Водійські посвідчення

### Видача вперше

**Етапи:**

1. **Медичний огляд** — довідка форми 083/о в закладі, що має ліцензію; усна оплата ~300–700 грн.
2. **Навчання в автошколі** — ліцензована школа; очний і теоретичний курс; категорії A, B, C, D, BE, CE тощо.
3. **Іспит в СЦ МВС** — теорія + практика (майданчик + місто).
4. **Видача посвідчення водія** — після успішного складання.

**Вимоги до віку:**

| Категорія | Мінімальний вік |
|---|---|
| **A** (мотоцикл) | 16 (до 125 см³) / 18 (без обмежень) |
| **B** (легковий автомобіль) | 18 |
| **C** (вантажний) | 18 (для роботи — 21) |
| **D** (автобус) | 24 |
| **BE, CE, DE** (з причепом) | 18 + категорія основного ТЗ |

### Обмін посвідчення водія

**Коли:** закінчення строку дії (для біометричних — 50 років, для нової категорії — обмін; зміна ПІБ; псування, втрата).

**Документи:**
- Паспорт (ID-картка).
- Попереднє посвідчення водія.
- Довідка форми 083/о (медична).
- РНОКПП.
- Платіжний документ.
- Для чоловіків 18–60 — військово-обліковий документ.

**Вартість обміну:** ~**459 грн** (адміністративний збір).

**Строк виготовлення:** 5–15 робочих днів (залежно від завантаженості).

### Посвідчення водія міжнародного зразка

**Діє разом із національним посвідченням** (а не замість нього). Видається на 3 роки.

**Вартість:** ~**290 грн** (адмінзбір).

### Відкриття нової категорії

- Пройти навчання в автошколі за новою категорією.
- Скласти іспит в СЦ МВС (теорія + практика для нової категорії).
- Отримати нове посвідчення з відкритою категорією.

### Втрата / викрадення посвідчення

- Заява в Національну поліцію (у разі викрадення) — протокол-талон.
- Заява в СЦ МВС про видачу дубліката.
- Медична довідка (не нова — довідка залишається у чинному вигляді).
- Адмінзбір — як при обміні.

### Іноземці / особи з посвідченнями інших держав

- **Посвідчення з країн Женевської / Віденської конвенції** — визнаються на території України протягом **60 днів** з моменту набуття права на проживання в Україні; після — обмін на українське.
- **Обмін без екзамену** — для посвідчень з ЄС / країн, з якими є двосторонні угоди.
- **З екзаменом** — для посвідчень з інших країн.

## 3. Реєстрація транспортних засобів

### Нова реєстрація ТЗ (вперше на території України)

**Ситуації:**
- Новий ТЗ (від дилера чи виробника).
- Імпортований ТЗ (після розмитнення).

**Документи:**
| Документ | Примітка |
|---|---|
| Паспорт (ID-картка) | — |
| РНОКПП | — |
| Документи власника-юр. особи | Статут, витяг ЄДРПОУ, довіреність, копії |
| **Договір купівлі-продажу** | Оригінал (бланковий або нотаріальний) |
| **Технічний паспорт виробника / закордонне свідоцтво** | Оригінал |
| **Митна декларація (МД)** | Для імпортних ТЗ |
| **Платіжні документи** про сплату акцизу, митних платежів | — |
| **Свідоцтво про звільнення від ввізного мита / акцизу** | За наявності пільги |
| **Документи на ідентифікаційний номер (VIN)** | На шасі, кузові |
| Квитанція про сплату зборів | — |
| Для чоловіків 18–60 — військово-обліковий документ | Обов'язково |

**Процедура:**
1. **Експертно-кримінальна перевірка** ТЗ — перевірка VIN, номерних агрегатів, ознак контрабанди чи розукомплектовки.
2. Після позитивного висновку — видача **свідоцтва про реєстрацію ТЗ** + державні номерні знаки (далі — ДНЗ).

**Строк:** 1 день (якщо немає зауважень експерта).

### Перереєстрація при купівлі-продажі

**Хто переоформлює:** покупець протягом **10 діб** від дати укладення договору купівлі-продажу.

**Документи:**
| Документ | Примітка |
|---|---|
| Паспорт + РНОКПП покупця | — |
| Паспорт продавця (обов'язковий при подачі бланкового ДКП) | Або нотаріально посвідчений ДКП, без присутності продавця |
| Свідоцтво про реєстрацію ТЗ (стара) | Здається |
| Платіжні документи — адмінзбір, пенсійний збір 1% / 3% | — |
| Для чоловіків 18–60 — військово-обліковий документ | Обов'язково |

**Адміністративний збір:** залежить від СЦ, типу послуг; стандартно **190–600 грн**.

**Пенсійний збір** (ЗУ № 400/97-ВР):
- **1%** — для легкових автомобілів, що **не вперше** реєструються в Україні (тобто, наступна зміна власника).
- **3%** — для легкових вартістю до 165 ПМ (прибл. < 500 000 грн) вперше в Україні.
- **4%** — 165–264 ПМ.
- **5%** — > 264 ПМ.
- **Звільнення**: для осіб з інвалідністю, УБД (перший автомобіль, раз на рік).

**Експертиза:** зазвичай вимагається; оплата — окремо, ~120–300 грн.

### Зняття з обліку

**Підстави:**
- Продаж ТЗ (хоча перереєстрація покупцем автоматично знімає з обліку продавця).
- Списання (зношене, після тотальної шкоди).
- Вивезення за кордон.

**Процедура:** заява власника + документи + адмінзбір.

### Тимчасовий реєстраційний талон

**Коли:** для виконання певних дій (ремонт, перегін, перевірка), якщо основне СРТЗ відсутнє.

**Строк дії:** 10 днів.

**Вартість:** ~200–400 грн.

### Дублікат свідоцтва про реєстрацію (СРТЗ)

**Коли:** втрата, псування, заповнення.

**Документи:**
- Заява (заповнюється в СЦ).
- Паспорт + РНОКПП.
- Квитанція про сплату.

**Вартість:** ~**219 грн**.

### Заміна номерних знаків

- При **втраті** / **псуванні** — заява, адмінзбір ~500 грн.
- При **бажанні обрати індивідуальний номер** («блатний») — окрема процедура, значно дорожче.

### Переобладнання ТЗ (зміна двигуна, кузова, кольору)

- Сертифікація переобладнання.
- Експертиза в СЦ МВС.
- Внесення змін до СРТЗ.

### Реєстрація ТЗ на іноземця

- **Постійна реєстрація** — іноземець повинен мати посвідку на тимчасове / постійне проживання.
- **Тимчасова реєстрація** — для дипломатів, міжнародних організацій.
- Документи: паспорт + посвідка + документи на ТЗ + сплата збору.

## 4. Військово-облікові вимоги періоду воєнного стану

Ст. 7-1 ЗУ «Про військовий обов'язок і військову службу» (доповнено ЗУ № 3633-IX від 18.05.2024):

**Для чоловіків 18–60 років** при зверненні за послугами СЦ МВС:
- Обов'язкова перевірка **військово-облікового статусу** (через реєстр «Оберіг» / «Резерв+»).
- **Ненаявність актуальних даних** — підстава для **відмови** в наданні послуги.
- **Стан СЗЧ / розшук** — гарантована відмова + направлення до ТЦК.

**Винятки** (коли відмова неприпустима):
- Видача ID-картки.
- Окремі категорії осіб з інвалідністю.

Див. також `ua:mobilization-defense` для оскарження відмов і спорів з ТЦК.

## 5. «Євробляхи» — пільгове розмитнення

ЗУ № 2612-VIII від 08.11.2018 — одноразова легалізація ТЗ з іноземною реєстрацією, ввезених до 24.11.2018.

**Поточний стан (2025 р.):**
- Термін пільг закінчився; ТЗ, що залишилися не розмитненими — нелегальні, без можливості реєстрації.
- СЦ МВС має списки «заборонених до реєстрації» VIN — перевіряти до купівлі.

## 6. Вартість основних послуг — шпаргалка

| Послуга | Типовий збір (2025 р.) |
|---|---|
| Обмін посвідчення водія | ~459 грн |
| Міжнародне посвідчення | ~290 грн |
| Перереєстрація ТЗ (легковий) | ~190–600 грн + пенсійний збір (1–5%) |
| Дублікат СРТЗ | ~219 грн |
| Зміна номерних знаків | ~500 грн |
| Перша реєстрація імпортного ТЗ | + експертно-кримінальна (120–300 грн) |
| Індивідуальний номерний знак | Від 5 000 грн і вище (аукціон) |

## 7. Оскарження

### Адміністративний порядок

**Підстава:** КАС України + ЗУ «Про адміністративні послуги» № 5203-VI (`5203-17`).

1. **Скарга керівнику СЦ МВС / регіонального сервісного центру** — 30 днів.
2. **Скарга до ГСЦ МВС** — як вища інстанція.
3. **Адмінпозов у порядку КАС** — 6 місяців з моменту, коли стало відомо про порушення (ст. 122 КАС).

### Судовий збір

- Адмінпозов (фіз. особа, немайновий): 0,4 ПМ — **1 211,20 грн** (2026).
- Для УБД, осіб з інвалідністю внаслідок війни — звільнення.

## 8. Типові помилки і підводні камені

- **Пропуск 10-денного строку** перереєстрації після купівлі — штраф (ч. 4 ст. 121 КУпАП, 425-595 грн), але головне — ризик, що продавець перереєструє ТЗ собі назад чи продасть третій особі.
- **Купівля ТЗ без перевірки hsc.gov.ua** — арешт / обтяження / розшук — перевіряти обов'язково перед оплатою.
- **Забування пенсійного збору** — при перереєстрації легкового ТЗ вартістю > 165 ПМ — до 5% від вартості, значна сума.
- **ТЗ «на посвідці іноземця»** — обмеження виникають при продажі / переоформленні — перевіряти статус посвідки.
- **Довіреність замість ДКП** — старий спосіб купівлі (до 2013 р.) — **не визнається** як підстава для переоформлення з 2013 р.
- **Фото 3×4 замість 3,5×4,5 (для посвідчення водія)** — у деяких СЦ вимагають; уточнювати.
- **Медична довідка старша 1 року** — не приймається; переробляти.
- **Не зареєстрований у «Резерв+» / «Оберіг»** — для чоловіків 18–60 р. — відмова в послузі. Зареєструватись **заздалегідь**.

## Кроки до звернення — шпаргалка

1. **Визначити послугу** на `hsc.gov.ua`.
2. **Зібрати документи** за переліком.
3. **Запис у чергу** онлайн: `https://hsc.gov.ua/e-queue`.
4. **Оплатити** адмінзбір + пенсійний збір (для ТЗ) — квитанція через «Дія» / банк.
5. **Прибути** в СЦ у визначений час.
6. **Пройти експертизу** (для ТЗ).
7. **Отримати** готовий документ / ДНЗ.

## Коли цей скіл доповнюється агентом / іншим скілом

- Для оскарження відмови СЦ МВС у судовому порядку — агент `ua:claim-drafter` (адмінпозов).
- Для аудиту договору купівлі-продажу ТЗ перед підписанням — скіл `ua:reviewing-vehicle-contract`.
- Для спору з ТЦК щодо військового обліку (після відмови СЦ МВС) — агент `ua:mobilization-defense`.
- Для оскарження штрафів ДАІ / патрульної поліції за КУпАП ст. 121, 122, 126 — агент `ua:response-drafter` (відгук) або `ua:appeal-drafter` (апеляція).
- Для отримання ID-картки чи паспорта громадянина — скіл `ua:applying-cnap-passport`.
