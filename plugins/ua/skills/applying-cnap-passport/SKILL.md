# applying-cnap-passport

Паспортні послуги в Україні надаються через **Центри надання адміністративних послуг (ЦНАП)** — міських, сільських, селищних рад, через **Державну міграційну службу (ДМС)** та **ДП «Документ»** (державне підприємство). В умовах воєнного стану — частково перенесено в онлайн-формат через застосунок **«Дія»** (паспорт у смартфоні, відмітки про реєстрацію), водночас випуск фізичних документів потребує особистого звернення (біометрія).

**Основні акти:**
- **ЗУ № 5492-VI «Про Єдиний державний демографічний реєстр та документи, що підтверджують громадянство України, посвідчують особу чи її спеціальний статус»** (`5492-17`, редакція актуальна).
- **ЗУ № 1382-IV «Про свободу пересування та вільний вибір місця проживання в Україні»** (`1382-15`).
- **Постанова КМУ № 302 від 25.03.2015** — видача ID-картки (паспорта у формі картки).
- **Постанова КМУ № 152 від 07.05.2014** — видача закордонного паспорта.
- **Постанова КМУ № 207 від 02.03.2016** — реєстрація / зняття з реєстрації місця проживання.
- **Постанова КМУ № 485 від 10.05.2022** — спрощене оформлення документів у період воєнного стану.

**Джерела:**
- **ДМС України**: `https://dmsu.gov.ua` — список територіальних підрозділів, записи, розрахунок зборів.
- **ДП «Документ»**: `https://pasport.org.ua` — альтернативний канал (прискорено, дорожче).
- **Портал «Дія»**: `https://diia.gov.ua`, мобільний застосунок — e-паспорт (офіційний аналог ID-картки і закордонного); довідка про реєстрацію.
- **zakon.rada.gov.ua** — акти.

## Актуальні параметри — отримати перед консультацією

| Параметр | Джерело | Як отримати | Fallback _(останній перевірений)_ |
|---|---|---|---|
| ПМ на 1 січня | ЗУ про Держбюджет | Значення ПМ — отримати через скіл `calculating-sudovyi-zbir` або WebSearch за бюджетним законом | 3 028 грн _(01.01.2026)_ |
| Збір за ID-картку | Постанова КМУ № 302 | WebFetch: `https://zakon.rada.gov.ua/laws/show/302-2015-%D0%BF` | 87 / 174 грн _(станом на 2025)_ |
| Збір за закордонний паспорт | Постанова КМУ № 152 | WebFetch: `https://zakon.rada.gov.ua/laws/show/152-2014-%D0%BF` | 682 / 1 034 грн _(станом на 2025)_ |

**Правила:**
1. **Fetch вдався** → використати отримане значення, зазначити джерело й дату.
2. **Fetch не вдався** → використати fallback. Попередити: «⚠ Збір використано зі станом на [дата]. Перевірте на zakon.rada.gov.ua або dmsu.gov.ua.»

## 1. ID-картка (паспорт громадянина України у формі картки)

**Хто може оформити:** громадяни України, яким виповнилося **14 років**.

**Типи звернень:**

| Ситуація | Періодичність / термін |
|---|---|
| **Вперше у 14 років** | Обов'язково протягом **30 днів** після 14-річчя (ч. 4 ст. 21 ЗУ 5492-VI) |
| **Обмін у 25 років, 45 років** | Обов'язковий обмін |
| **Втрата, знищення, зіпсування** | Протягом 30 днів (бажано) |
| **Зміна прізвища / імені / по батькові** | 30 днів |
| **Помилки / технічні недоліки** | Без обмеження терміну |

### Документи для ID-картки — вперше у 14 років

| Документ | Примітка |
|---|---|
| Свідоцтво про народження | Оригінал; за його відсутності — повторно отримати в РАЦС |
| Паспорт одного з батьків або інший документ, що посвідчує особу законного представника | — |
| Довідка про реєстрацію місця проживання або витяг з реєстру територіальної громади | — |
| Довідка про присвоєння РНОКПП (ідентифікаційного коду) | Якщо на батьківське подання від ДФС |
| Оплата адміністративного збору | Див. розділ про збори |
| 1 фотокартка | Якщо обладнання ЦНАП не робить фото на місці (рідко — зазвичай біометрія з камери на місці) |

### Документи для обміну ID-картки у 25 / 45 років

- Попередній паспорт (ID-картка або паспорт у формі книжечки — підлягає обміну).
- Реєстраційний номер облікової картки платника податків (РНОКПП).
- Квитанція про сплату адміністративного збору.
- Довідка про реєстрацію місця проживання.

### Документи при втраті / викраденні

- **Заява про втрату**, зареєстрована в підрозділі Національної поліції (талон-повідомлення) — у разі викрадення.
- При втраті — заява в довільній формі до ЦНАП / ДМС.
- Інші документи (свідоцтво про народження, РНОКПП, довідка про реєстрацію).

### Адміністративний збір за ID-картку

| Підстава | Сума |
|---|---|
| **Вперше у 14 років** | Безкоштовно (пільга) |
| Обмін у 25 / 45 років у встановлений строк | **87 грн** _(fallback; станом на 2025)_ |
| Повторне оформлення (втрата, псування) | **174 грн** _(fallback; станом на 2025)_ |
| Термінове оформлення (8 р.дн.) | + додатковий збір (прискорено, до ~2x) |

**Строки виготовлення:**
- **Стандартно** — **20 робочих днів** з дати подання заяви.
- **Терміново (через ДП «Документ»)** — **7–10 робочих днів**, підвищений тариф.

## 2. Закордонний паспорт (біометричний)

**Регулюється:** постанова КМУ № 152 від 07.05.2014.

### Типи звернень

| Ситуація | Примітки |
|---|---|
| Вперше | З народження |
| Обмін (строк закінчується / сплинув) | Поточний строк дії біометричного паспорта — 10 років для дорослих, 4 роки для дітей до 16 років |
| Втрата, зіпсування | Документально підтверджена |
| Зміна ПІБ, громадянства | — |
| Зовнішність значно змінилась (післяопераційна / травма) | — |

### Документи

| Документ | Примітка |
|---|---|
| ID-картка (для осіб ≥ 14 років) | Обов'язково |
| Свідоцтво про народження | Для дітей < 14 років |
| Попередній закордонний паспорт | За наявності (для обміну) |
| Довідка про РНОКПП | Якщо не вказано в ID-картці |
| Згода другого з батьків | Для дитини (нотаріально посвідчена, у разі відсутності — на підставі рішення суду) |
| Військово-обліковий документ | Для чоловіків 18–60 років — обов'язково (ч. 6 ст. 23 Закону № 5492-VI); з 01.07.2024 — вимоги посилено |
| Фотокартка | Біометрія на місці |
| Квитанція про сплату | — |

### Адміністративний збір за закордонний паспорт

| Підстава | Сума |
|---|---|
| **Стандартно (20 р.дн.)** | **682 грн** _(fallback; станом на 2025)_ |
| **Терміново (7 р.дн., ДП «Документ»)** | **1 034 грн** _(fallback; станом на 2025)_ |
| **За кордоном (консульство)** | Консульський збір — окремо; див. `ua:applying-consular-procedures` |
| Дитина до 16 років | Подвійно дешевше в деяких категоріях |

## 3. Військово-облікові вимоги під час воєнного стану

З 18.05.2024 (ЗУ № 3633-IX) — для чоловіків віком **18–60 років** отримання / оновлення документів у ЦНАП / ДМС / ДП «Документ» дозволяється **лише за умови перебування на військовому обліку** та наявності оновлених даних у реєстрі «Оберіг» / застосунку «Резерв+».

**Практика:**
- При поданні заяви ЦНАП перевіряє військово-обліковий статус.
- **Ненаявність** актуальних даних — підстава для відмови в наданні послуги (окрім ID-картки, яка є обов'язковим документом — тут ЦНАП направляє до ТЦК).
- **Вже виданий паспорт** не анулюється; проблема — при наступному оновленні.

Докладніше — див. `ua:mobilization-defense` для спорів з ТЦК.

## 4. Реєстрація / зняття з реєстрації місця проживання

**Основа:** ЗУ № 1382-IV + постанова КМУ № 207 від 02.03.2016.

### Реєстрація нового місця проживання

**Термін подачі:** **30 днів** з моменту зміни місця проживання (ч. 1 ст. 6 ЗУ 1382-IV).

**Куди звернутись:** ЦНАП за новим місцем проживання.

**Документи:**
| Документ | Примітка |
|---|---|
| Паспорт (ID-картка) | — |
| Свідоцтво про народження | Для дітей |
| **Документ про право власності** на житло (витяг з ДРРП / договір купівлі-продажу / дарування / спадкування) | Оригінал + копія |
| **Або** письмова згода власника житла (нотаріально посвідчена, якщо власник не може бути присутній особисто) | — |
| Військово-обліковий документ | Для чоловіків 18–60 років |
| Заява встановленої форми | Заповнюється в ЦНАП |

**Адміністративний збір:** **13,60 грн** (0,0085 ПМ) _(fallback; станом на 2025; залежить від ПМ)_.

**Строк реєстрації:** **у день звернення**.

### Зняття з реєстрації

**Куди:** ЦНАП за старим місцем проживання. Часто виконується автоматично при реєстрації нового місця, тому не вимагає окремого звернення.

**Зняття з реєстрації дитини молодше 14 років:**
- Згода другого з батьків (нотаріально посвідчена), якщо не подає обидва;
- Нове місце реєстрації не має бути значно погіршено.

### Довідка про реєстрацію місця проживання (форма 3)

**Форма 3** — довідка для оформлення документів, звернення до державних органів, банків.

**Отримати:**
- Через ЦНАП (друкується на місці; 1 день).
- **Онлайн через «Дія»** — без звернення до ЦНАП, якщо реєстрацію вже внесено в реєстр територіальної громади.

**Вартість:** 13,60 грн _(fallback; станом на 2025)_ (через ЦНАП) / безкоштовно через «Дія».

## 5. E-паспорт у «Дії»

**Підстава:** ЗУ № 1135-IX «Про публічні електронні реєстри», ЗУ № 1955-IX «Про електронні довірчі послуги», ст. 14-1 ЗУ № 5492-VI.

**Статус:** офіційний аналог ID-картки та закордонного паспорта на території України (ст. 14-1 ЗУ № 5492-VI, постанова КМУ № 911 від 18.08.2021).

**Обмеження:**
- **Не визнається за кордоном** — для виїзду потрібен фізичний біометричний паспорт.
- **Не приймають окремі установи** (банки при відкритті рахунку, приватні нотаріуси у деяких ситуаціях) — у залежності від їхніх внутрішніх правил.

**Активація:** через застосунок «Дія» після першого отримання ID-картки або закордонного паспорта, підтвердження банківським BankID / Mobile ID.

## 6. Особливі ситуації

### Реєстрація дитини

- Одночасно при реєстрації в РАЦС (до 30 днів після народження).
- ID-картка видається з 14 років; до того — свідоцтво про народження + паспорт одного з батьків для подорожей.

### ВПО (внутрішньо переміщені особи)

- Довідка ВПО (форма 1ВПО) оформлюється в ЦНАП — автоматично при переїзді з окупованих / тимчасово непідконтрольних територій. Видається безкоштовно.
- Для довідки ВПО паспорт / ID-картка обов'язкові; у разі відсутності — відновлення через ДМС.

### Паспорти, видані на окупованій території до окупації

- Діючі, якщо не минув строк.
- Обмін відбувається через континентальний ЦНАП — без додаткових вимог.

### Паспорти, видані окупаційною владою рф / «днр» / «лнр»

- **Не визнаються** в Україні (ЗУ № 2268-IX, постанова КМУ про перелік документів).
- Для отримання українських документів — процедура встановлення особи.

### Поновлення громадянства

- Окрема процедура (ЗУ № 2235-III «Про громадянство України»); після поновлення — оформлення паспорта.

### Громадяни України з подвійним громадянством

- Українське громадянство пріоритетне (ч. 1 ст. 2 ЗУ 2235-III); у державі Україні особа розглядається як українець.
- Оформлення ID-картки / закордонного паспорта — звичайний порядок.

## 7. Оскарження відмови

### Адміністративний порядок

**Підстава:** ст. 15 ЗУ 5492-VI + КАС України.

1. **Скарга до ДМС / територіального управління** — 30 днів від дня, коли стало відомо про порушення.
2. **Скарга до суду адміністративної юрисдикції** — за місцезнаходженням органу або за місцем проживання позивача (ст. 25 КАС):
   - **6 місяців** з дня, коли стало відомо про порушення (ст. 122 КАС);
   - **3 місяці**, якщо використано досудовий порядок.

### Судовий збір

- **Адмінпозов немайновий (фізична особа):** 0,4 ПМ = **1 211,20 грн** (на 2026 р., ПМ = 3 028 грн).
- Для пільгових категорій (УБД, особи з інвалідністю внаслідок війни) — **звільнення** за ст. 5 ЗУ «Про судовий збір».

## Типові помилки і підводні камені

- **Ігнорування строку 30 днів** після 14-річчя для ID-картки — штраф не передбачено, але пізніше мати ID стає ускладненим (не можна оформити закордонний паспорт без ID після 14, не можна отримати РНОКПП).
- **Відсутність документа про право на житло** під час реєстрації — без нього неможливо. Якщо орендоване житло — нотаріально посвідчена згода власника.
- **Ігнорування військово-облікових вимог** (для чоловіків 18–60 р.) — відмова в наданні послуги; дорога до ТЦК.
- **Мобільне фото замість біометрії** — не приймається для ID/закордонного; тільки на місці.
- **Оформлення через посередника** (не ДП «Документ») — часто нелегально, з ризиком втрати документів.
- **Забування РНОКПП** — багато процедур вимагають; для його отримання — окреме звернення до ДПС (окремо).
- **Копії паспортів закордонні / ID — не заверені** — для різних установ часто потрібна нотаріально посвідчена копія.

## Кроки для звернення — швидка шпаргалка

1. **Записатись онлайн** — `dmsu.gov.ua/services/e-queue` (ДМС) або `pasport.org.ua` (ДП «Документ»).
2. **Підготувати документи** згідно переліку.
3. **Оплатити** адміністративний збір (квитанція — через «Дія» / банк).
4. **Прийти** в записаний день; здати документи + біометрія.
5. **Отримати** розписку про прийняття заяви.
6. **Перевірити готовність** через сервіс `dmsu.gov.ua` або SMS.
7. **Забрати** готовий документ.

## Коли цей скіл доповнюється агентом / іншим скілом

- Для оскарження відмови ДМС / ЦНАП у судовому порядку — агент `ua:claim-drafter` (адмінпозов).
- Для допомоги в установленні особи без паспорта (колишні громадяни, народжені до 1991 в СРСР) — агент `ua:request-drafter` (звернення до ДМС).
- Для оформлення документів за кордоном — скіл `ua:applying-consular-procedures`.
- Для спорів з ТЦК про військово-обліковий статус при отриманні паспорта — агент `ua:mobilization-defense`.
- Для оформлення ВПО (не документа, а соціального статусу) — рекомендовано окремий скіл (у планах).
