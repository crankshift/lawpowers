# checking-martial-law-overrides

З 24.02.2022 в Україні діє правовий режим воєнного стану. Численні норми модифіковані спеціальним регулюванням. Застосування довоєнної норми без перевірки спецрегулювання — ризик серйозної помилки.

## Базове регулювання

| Акт | zakon.rada ID |
|---|---|
| ЗУ «Про правовий режим воєнного стану» | 389-19 |
| ЗУ «Про оборону України» | 1932-12 |
| ЗУ «Про мобілізаційну підготовку та мобілізацію» | 3543-12 |
| ЗУ «Про організацію трудових відносин в умовах воєнного стану» | 2136-20 |
| ЗУ «Про основи національного спротиву» | 1702-20 |
| ЗУ «Про внесення змін… у зв'язку з веденням воєнного стану» | низка законів |
| Укази Президента про введення/продовження ВС | щоразу новий |
| Постанови КМУ 2022-2026 | численні |

Дія воєнного стану продовжується окремими указами Президента (затверджуються ЗУ) — завжди перевіряти, на який період діє актуальний указ.

## Сфери типових відхилень

**Процесуальні строки (ЦПК, ГПК, КАС, КПК):**
- Продовження/зупинення строків за певних умов.
- Електронне подання — розширене застосування.
- Розгляд у режимі відеоконференції — спрощено.

**Позовна давність:**
- Спеціальні правила для осіб із окупованих/прифронтових територій.
- Зупинення строків на період неможливості звернення до суду (ст. 263 ЦК — про зупинення внаслідок непереборної сили).

**Нотаріат і реєстри:**
- Зупинення/обмеження певних реєстраційних дій у різні періоди.
- Особливості посвідчення правочинів.
- Режим роботи Державних реєстрів (ДРРП, ЄДР, ЄДРПОУ) у різні фази ВС.

**Трудові відносини:**
- Розширені підстави звільнення / призупинення трудового договору (ЗУ 2136-20).
- Спецправила для мобілізованих (збереження робочого місця, середнього заробітку).
- Скорочення строку попередження про звільнення.

**Майнові права:**
- Обмеження відчуження окремих категорій майна.
- Валютні обмеження НБУ (особливо транскордонні операції).
- Особливості спадкування (продовження строків прийняття спадщини).

**Судочинство:**
- Передача територіальної підсудності з окупованих / прифронтових територій (ст. 147 ЗУ «Про судоустрій і статус суддів» + спец. розпорядження Голови ВС).
- Онлайн-розгляд.

**Податки:**
- Тимчасові мораторії, спецпільги, відстрочення.
- Особливості перевірок.

**Мобілізація:**
- Бронювання працівників критичних підприємств (постанови КМУ).
- Обмеження на виїзд, особливості документування.

## Workflow

1. **Перед застосуванням норми** про строки, реєстрації, трудові/майнові правила — запитати себе: «Чи не модифіковано це воєнним станом?».
2. **Пошук спецрегулювання** через zakon.rada:
   - `"в умовах воєнного стану" {тема}`
   - `"на період дії воєнного стану" {тема}`
   - Перевірити останні зміни до профільного кодексу/закону на предмет «Прикінцеві положення. п. ... У період дії воєнного стану...».
3. **Актуальність** — багато ранніх постанов КМУ 2022 року вже скасовано або замінено.
4. **Дата правовідносин** — режим змінювався в часі. Що діяло у березні 2022 ≠ що діє у 2026.
5. **Територія** — режим відрізняється на окупованих, прифронтових, тилових територіях. Окремі акти КМУ визначають переліки територій.

## Common mistakes

- **Застосування довоєнних процесуальних строків** без перевірки спеціальних правил.
- **Посилання на скасовану постанову КМУ** 2022 року, яка вже не діє.
- **Узагальнення «під час воєнного стану…»** без вказівки конкретної дати — норми змінювалися.
- **Ігнорування територіального фактора** — правила різні для різних територій.
- **Припущення, що ВС діє автоматично і безстроково** — насправді продовжується окремими указами; перевіряти актуальний указ на дату.
- **Плутання ВС з надзвичайним станом** — різні правові режими з різним регулюванням.
