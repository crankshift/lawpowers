# legal-ua

Плагін Claude Code для роботи з українським законодавством і правничими документами. Містить спеціалізованих субагентів під конкретні юридичні задачі та скіли для ефективної роботи з офіційними джерелами (zakon.rada, ЄДРСР, сайти ВС/КСУ).

> Робочий простір для юриста-практика. Усі документи, відповіді та шаблони — українською.

## Встановлення

Плагін `legal-ua` можна встановити двома способами:
- через **Claude Desktop App** (macOS/Windows) — у UI налаштувань;
- через **Claude Code CLI** (термінал) — командами `/plugin`.

### Варіант 1. Claude Desktop App (macOS/Windows)

1. Відкрити **Settings → Extensions → Plugins**.
2. Перейти на вкладку **Personal**.
3. Натиснути «**+**» поруч із «Local uploads».
4. У меню обрати **Add marketplace**.
5. Вказати джерело: `crankshift/legal-ua` (або повний URL `https://github.com/crankshift/legal-ua`).
6. Після додавання маркетплейсу відкрити плагін `legal-ua` у списку і натиснути «+» для встановлення.

### Варіант 2. Claude Code CLI (термінал)

Запустити:

```bash
claude
```

У відкритій сесії ввести по черзі:

```
/plugin marketplace add crankshift/legal-ua
/plugin install legal-ua@legal-ua
/reload-plugins
```

### Варіант 3. Локально для розробки / приватного репозиторію

```bash
git clone https://github.com/crankshift/legal-ua.git
claude --plugin-dir ./legal-ua
```

Для приватного репо через Git URL:

```
/plugin marketplace add https://github.com/crankshift/legal-ua.git
/plugin install legal-ua@legal-ua
```

### Перевірка встановлення

Незалежно від клієнта:

- `/plugin` → вкладка **Installed** — плагін `legal-ua` у списку.
- `/agents` — субагенти з префіксом `legal-ua:` (наприклад, `legal-ua:claim-drafter`, `legal-ua:raport-drafter`).
- Скіли активуються автоматично за контекстом: згадка «судовий збір» тригерить `legal-ua:calculating-sudovyi-zbir`; «рапорт на відпустку» — `legal-ua:raport-drafter`.

### Оновлення

```
/plugin marketplace update legal-ua
/reload-plugins
```

Або увімкнути auto-update у `/plugin` → **Marketplaces** → `legal-ua` → **Enable auto-update**.

## Що всередині

### Субагенти (`agents/`)

| Агент | Призначення |
|---|---|
| `claim-drafter` | Позовні заяви (ЦПК/ГПК/КАС), зустрічні позови, уточнення вимог, розрахунок судового збору |
| `response-drafter` | Відзиви, заперечення, пояснення — документи з боку відповідача |
| `appeal-drafter` | Апеляційні та касаційні скарги; правові позиції ВС |
| `motion-drafter` | Процесуальні клопотання (забезпечення позову, витребування доказів, експертиза, відводи, строки) |
| `legislation-analyst` | Аналіз чинного законодавства, тлумачення норм, перевірка редакції на дату |
| `legal-memo` | Правові висновки, меморандуми, аналіз перспектив спору |
| `request-drafter` | Адвокатські запити (ст. 24 ЗУ «Про адвокатуру») та запити на публічну інформацію |
| `contract-drafter` | Складання і аналіз ЦП/господарських договорів, перевірка на ризики |
| `debt-collector` | Стягнення заборгованості: претензія → позов → виконавче (ст. 625 ЦК + пеня) |
| `enforcement-agent` | Виконавче провадження: заяви, скарги на дії ДВС/ПВВ |
| `arbitration-agent` | МКАС/МАК, ICC/LCIA/SCC/SIAC/HKIAC/VIAC, ad hoc UNCITRAL, інвестарбітраж; визнання і скасування рішень у держсуді |

#### Окремий блок: військовослужбовці ЗСУ

| Агент | Призначення |
|---|---|
| `raport-drafter` | Рапорти військовослужбовця ЗСУ — відпустка, ВЛК, переведення, звільнення за ст. 26 ЗУ 2232-XII, виплати, УБД, контракт, шлюб через відеозв'язок |
| `vlk-appeal` | Оскарження висновків військово-лікарської комісії — ієрархічне (вища ВЛК) і судове (адмінпозов у КАС) |
| `military-social-benefits` | Виплати і пільги: бойові (КМУ № 168), фронтова надбавка, ОГД при пораненні/загибелі, виплати родинам полонених/зниклих безвісти (ЗУ 3995-IX), статус УБД, пенсії |
| `mobilization-defense` | ТЦК, бронювання (з 01.01.2026 мін. ЗП 21 617,50 грн), відстрочки, оскарження штрафів за ст. 210 КУпАП, адмінпозови до ТЦК |
| `szch-defense` | Повернення з СЗЧ і кримзахист за ст. 407-408 КК; звільнення від відповідальності за ч. 5 ст. 401 КК (Закон № 3902-IX) |

| `searching-edrsr` | Пошук судової практики в ЄДРСР, структура номера справи |
| `calculating-sudovyi-zbir` | Розрахунок судового збору, ставки в ПМ, пільги |
| `citing-ukrainian-law` | Формат цитування НПА, рішень ВС/КСУ/ЄСПЛ |
| `determining-ua-jurisdiction` | Визначення підсудності (цивільна / господарська / адміністративна) |
| `checking-pozovna-davnist` | Строки позовної давності, перебіг, зупинення/переривання |
| `checking-martial-law-overrides` | Перевірка спецрегулювання періоду воєнного стану |

| `applying-new-york-convention` | NYC 1958 при визнанні/виконанні арбітражних рішень в Україні |

| Скіл | Коли застосовується |
|---|---|
| `military-statute-refs` | Таблиця ID ключових військових НПА (статути, ЗУ 2232-12, 2011-12, 3551-12, накази МО, постанови КМУ 168/419/560/76) |
| `calculating-military-payments` | Формули розрахунку бойових, фронтової надбавки, ОГД, матдопомоги, пенсій; прецедент ВС № 280/8933/24 |

| `szch-decriminalization` | Чек-лист за ч. 5 ст. 401 КК (Закон № 3902-IX): перший випадок + рапорт + згода командира + явка у 72 год |

## Принципи роботи

- **Дослівність цитат** — норма наводиться в точній редакції, чинній на вказану дату.
- **Обов'язкові посилання** — кожна правова позиція з посиланням на статтю + джерело + дату звірки.
- **Актуальність** — українське законодавство змінюється часто (особливо за воєнного стану); перед використанням норми перевіряти чинну редакцію на zakon.rada.gov.ua.
- **Не вигадувати практику** — номери справ, дати постанов, цитати з рішень — лише з ЄДРСР / сайту ВС.
- **Персональні дані** — у шаблонах плейсхолдери (`[ПІБ]`, `[РНОКПП]`, `[адреса]`), без реальних даних клієнтів.
